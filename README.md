# Hotel Booking App

A full-stack hotel booking application that allows users to reserve rooms and receive instant email confirmations.

## 🚀 Overview

This project demonstrates a complete booking workflow where users can select room details, submit their booking, and automatically receive a confirmation email. It focuses on backend integration, API handling, and email automation.

## ✨ Features

* Room booking with user details
* Dynamic form handling
* Automated email confirmation system
* Backend API for handling booking requests
* Secure environment variable management

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Backend:** Node.js, Express.js
* **Email Service:** Nodemailer (SMTP - Gmail)

## 📌 Key Highlights

* Implements real-world email automation workflow

* Uses environment variables to protect sensitive credentials
* Designed as a deployable full-stack project

## 📬 Functionality Flow

1. User fills out booking form
2. Data is sent to backend API
3. Server processes booking request
4. Confirmation email is sent to user
